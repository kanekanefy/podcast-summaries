# Meltem Kuran Berkowitz - Lenny's Podcast 摘要

## 嘉宾简介
Meltem Kuran Berkowitz是Deel的增长负责人,Deel是有史以来增长最快的SaaS公司之一,从0收入到3亿美元ARR仅用了3年时间,同时保持EBITDA盈利。她在公司仅有第19-20名员工时加入,领导了增长团队从早期到现在的全部工作,包括付费广告、产品营销、内容、社区、品牌等。

## 核心议题
1. **低成本增长渠道**: 如何利用社区、SEO、内容等低成本方式启动增长
2. **SEO策略**: 如何构建高效的SEO运营体系
3. **团队建设**: 如何在早期构建高绩效的增长团队
4. **文化塑造**: 如何建立"Deel速度"和乐观主义文化

## 核心洞察

### 1. 低成本增长的核心哲学
**核心洞察**: "人们不想被推销,他们想要问题得到解决"

- **问题-答案匹配模式**: 在Reddit、Quora、Twitter等地方寻找人们正在提问的地方,提供有价值的答案
- **社区策略**: 在subreddit中设置关键词追踪,当人们提问时主动回答
- **价值优先**: 先回答问题,再自然地介绍你的解决方案
- **长期复利**: 你提供的答案会长期存在于这些平台上,持续为你带来流量

**比喻**: "就像在派对上,如果你想交朋友,你得先成为一个人们真正想交谈的人"

### 2. SEO的红绿灯系统
**核心洞察**: "Google搜索是否结束"是检验内容质量的金标准

**红绿灯框架**:
- **绿灯**: 高意向+高流量 → 这是理想客户在寻找解决方案
- **黄灯**: 50/50意向 → 可能是潜在客户,但时机未到
- **红灯**: 低意向 → 只是大学学生在写文章,永远不会成为客户

**操作要点**:
1. 找到最多700个相关关键词
2. 按流量量级排序
3. 评估每个关键词的搜索意图
4. 从绿灯最高流量开始,依次向下

### 3. 产品质量是一切的基础
**核心洞察**: "如果你是早期B2B公司,有6个销售人员和2个工程师,你的产品可能不会太好"

- **团队配置判断**: 早期公司大部分应该是产品和工程师
- **产品体验**: 网站加载速度超过4秒就无法成功运行付费广告
- **过度承诺风险**: 在Deel早期,他们甚至需要"调低"营销信息,因为产品太好以至于人们不相信

### 4. 增长团队的演变
**早期结构** (功能型):
- 第一个招聘:产品营销负责人
- 第二个招聘:文案/创意负责人
- 第三个招聘:数据分析师(在仅有35人时)

**现在结构** (功能+区域):
- **功能团队**: 产品营销、内容、社区、活动、付费广告、品牌
- **区域团队**: 不同重点地区的本地化营销专家

**关键原则**: 将技术角色集中管理,而不是分散到各地区,这样最佳实践的学习和技能提升更快

## 心法与原则

### Deel的三大文化价值观

1. **Deel速度**:
   - 客户问题不会等两周,必须在24小时内得到回答
   - 产品构建速度是竞争对手的十分之一

2. **默认乐观**:
   - 不是问"这为什么不行",而是问"我认为这行的通的四个理由是什么?现在让我们看看风险并解决它们"

3. **真心在乎客户**:
   - 这不是关于软件和平台,而是关于企业支付给人类让他们生活,业务能够继续增长

**"小手"概念**: 无论你在组织中处于什么位置,都需要愿意做小事和琐碎工作,不回避

### 增长哲学的演变

**早期** (80-90%非付费渠道):
- Reddit/Quora社区回答
- SEO内容
- 合作伙伴关系

**现在** (50%非付费渠道):
- 上述渠道继续
- 但绝对数量在增长
- 百分比下降是因为其他渠道增长更快

## 实战手册

### 1. 早期增长投资优先级

**步骤1**: 基础设施检查
- 你有网站吗?
- 它快吗?
- 搜索引擎知道它的存在吗?

**步骤2**: 可发现性
- 人们能找到它吗?
- 如果不能,你需要写内容让人们能够找到它

**步骤3**: 付费广告
- 只有在以上问题都得到答案后,才考虑:我有钱吗?
- 我能把它放在一些付费广告后面确保人们来到我的网站吗?

**原则**: "回到基础,先在核心建立良好体验,然后逐步扩展"

### 2. 社区增长战术

**Reddit策略**:
1. 设置关键词提醒工具
2. 当相关问题出现时主动回答
3. 提供真正的价值,不仅仅是"看看我们的网站"
4. 认识到受众上限:1000人的subreddit可能只能赢得10%的人

**开放社区**:
- Y Combinator等孵化器社区
- Discord、Slack专业群组
- 创始人连接社区

**关键**: 先添加价值,否则会被踢出

### 3. SEO内容运营

**发布节奏**:
- 过去:每周10篇新文章
- 现在:每周5篇新文章+5篇更新文章

**团队结构** (8人):
- 1位内容总监(Deel第2-3号员工)
- 1位运营负责人(管理自由职业者、briefs、事实核查)
- 不同产品线的内容专家
- 新媒体团队(视频、教育)

**工具**:
- Clearscope:内容优化和评分
- 各种SEO工具确保语言足够简单(五年级阅读水平)

### 4. 付费广告成功要素

**两大支柱**:

1. **优化层面**:
   - 技术性设置出价
   - 预算管理
   - 确保不超支

2. **信息传递**:
   - 创意疲劳是真实存在的
   - 广告在Instagram、Twitter、Google上都要有意义
   - 每月更新创意(Google广告除外)

**关键指标**: 不仅看带来的线索数量,还要看:
- 多少实际成为客户
- 一年后实际赚了多少钱
- 回本周期

**长尾策略**:
- 30%的线索流来自长尾平台
- 评审网站、小型出版物、通讯广告、播客广告
- 单个可能不重要,但加起来很可观

### 5. 合作伙伴关系

**关键洞察**: "你与谁合作很重要"

**两类合作伙伴**:
1. **共享受众**: 这是一个不错的起点
2. **信任资源**: 人们是否视这个合作伙伴为可信的解决方案来源

**成功案例**: 风险投资合作伙伴
- 当公司获得新资金时,会问VC:"你的其他投资组合公司用什么平台?你信任并推荐什么平台?"
- 人们会向VC寻求这类问题

**失败案例**: 其他科技公司虽然有相同的受众,但人们永远不会向他们询问这个问题

### 6. 雇佣原则

**面试问题**: "你愿意承诺的KPI是什么?"
- 如果只愿意承诺线索数字,那不够好
- 需要能够承诺到底层收入目标

**案例研究测试**:
- 给候选人$0、$10,000、$100,000三种预算
- 看他们如何思考和承诺不同支出水平的KPI
- 如果有人说"$0只能做社交媒体监控,不会有太大效果",他们可能不习惯承诺到底层指标

**"小手"测试**:
- 无论级别如何,都要愿意做最微小的工作
- 如果第一个答案是"我会为此组建团队",可能不太适合早期阶段

## 反常识视角

1. **早期不要做品牌认知活动**
   - B2B企业应该先挖掘漏斗底部
   - 需要至少6-8个月挖掘准备转化的人群
   - 认知广告是"你想做的很酷的事情",但往往浪费时间

2. **SEO不适合所有公司**
   - 如果人们在寻找解决方案,应该做SEO
   - 如果你是卖口红的DTC公司,可能不应该在SEO上投入太多
   - 人们不会去Google找口红,他们会去Instagram、网红

3. **增长团队应该有收入目标**
   - 这就是为什么叫"growth"而不是"marketing"
   - 当然跟踪线索和SQL,但这些只是领先指标
   - 最终成功不是靠这些衡量的

4. **自动化社区互动是大忌**
   - 从来不是自动化的
   - 今天仍然不是
   - 团队每个人都可以访问这些渠道
   - 有时Twitter上一个问题会有3个Deel团队成员回答

## 花絮与细节

**个人故事**:
- 在伊斯坦布尔探亲时,和丈夫石头剪刀布争夺书桌使用权
- 输了之后用熨衣板当书桌,背后是博斯普鲁斯海峡的美景
- 在Twitter分享后走红,展示远程工作的现实
- 在加拿大加入Deel,疫情期间工作了一年半后才第一次见到同事
- 在里斯本会议上第一次见到团队成员,当时公司收入已经超过5000万美元

**IRS链接故事**:
- 在前公司(Bench Accounting)时,团队创建了PPP贷款申请资源
- 因为质量太高,IRS网站链接到这篇文章:"如果有问题,查看这篇文章"
- 在人们申请PPP的几周内带来了巨大增长

**企业文化花絮**:
- 在达到10亿美元估值之前忘记创建周边商品
- 当要在纳斯达克拍照时,才意识到需要T恤
- 不得不派人跑到纽约商店打印logo

**远程文化**:
- 从第一天起就是远程优先
- 人们可以在WeWork和联合办公空间工作,但大多数选择在家
- 人们在她正式见面之前就见过她的宠物和伴侣
- 不需要以某种方式展示自己或为会议打扮
