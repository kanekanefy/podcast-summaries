# Nilan Peiris - Lenny's Podcast 播客摘要

## 嘉宾简介

Nilan Peiris 是 Wise（前 TransferWise）的首席产品官（CPO），在那里工作了超过11年，基本上是从公司开始的阶段。

Wise 的核心业务是让国际汇款变得极其简单和便宜。Lenny 是 Wise 的每周活跃用户，因为产品太好了，Wise 主要通过口碑增长。大约 **70% 的增长来自口碑**。

Lenny 认为 Nilan 关于如何推动口碑增长的建议是他听过最战术性和有用的建议。

## 核心议题

本次对话的核心议题是**如何推动口碑增长**（Word of Mouth Growth）。

Nilan 分享了 Wise 如何实现 70% 的口碑增长率，以及具体如何让口碑如此成功。这是任何创始人都希望拥有的增长类型。

## 核心洞察

### 1. Wise 的商业模式

**Wise 做什么**：
- 解决跨境货币转移的问题
- 跨境转移很慢、很贵、很难做
- Wise 通过三个产品解决这个问题：
  1. **汇款产品**：他们开始的产品
  2. **账户**：像国际银行解决方案
  3. **企业产品**：将底层基础设施嵌入银行和日常产品

**关键指标**：
- 每月转移约 120 亿美元
- 年增长率 30-40%
- 平均收费约 0.65%
- 已经盈利超过4年
- EBITDA 利润率 20%

### 2. 口碑增长的力量

**70% 的增长来自口碑**：
- 这是令人惊讶的高比例
- 大多数公司渴望这样的增长
- Wise 做了一些特别的事情来实现这一点

**为什么口碑如此强大**：
- 可信度高（来自朋友推荐）
- 获客成本低
- 留存率更高
- 增长可持续

### 3. 将营销融入产品

**Nilan 的洞察**：
- "营销和获取客户是产品的一部分"
- "我们应该将其无缝地缝合到体验中"
- "这样我们就能更有效地做这件事"

**实践**：
- 营销不是独立的功能
- 而是产品体验的有机组成部分
- 用户在产品使用过程中自然地分享和推荐

### 4. 产品体验驱动增长

**关键原则**：
- 产品太好了，用户自然想告诉朋友
- 不需要强制性的推荐计划
- 真实的用户满意度驱动分享

**Wise 的做法**：
- 专注于产品体验
- 让汇款变得简单、快速、便宜
- 用户因为满意度而推荐

## 心法与原则

### 核心心法

1. **产品是增长的第一驱动力**
   - 如果你把产品做得足够好
   - 用户会自然地告诉朋友
   - 这是最可持续的增长方式

2. **口碑是可设计的**
   - 口碑不是运气
   - 可以通过产品设计来鼓励
   - 需要有意识地构建产品

3. **营销应该融入产品**
   - 不是事后添加
   - 应该是产品体验的一部分
   - 无缝集成

4. **透明度建立信任**
   - Wise 展示真实的费用
   - 不隐藏成本
   - 这建立了信任，促进了口碑

### 关键原则

1. **简单优于复杂**
   - 让汇款变得极其简单
   - 用户不需要理解复杂的金融
   - 简单的体验驱动推荐

2. **价格透明**
   - 展示真实的费用
   - 与银行比较
   - 这创造了"哇"的时刻

3. **速度和可靠性**
   - 快速转账
   - 可靠的服务
   - 超出预期

4. **持续优化**
   - 不断改进产品
   - 倾听用户反馈
   - 快速迭代

## 实战手册

### 如何推动口碑增长

#### Nilan 的战术建议：

1. **创造"哇"时刻**
   - 超出用户预期
   - 让他们说"我不敢相信这么简单"
   - 这些时刻驱动分享

2. **透明定价**
   - 展示真实费用
   - 与传统方式比较
   - 让用户感觉到"我发现了秘密"

3. **简化体验**
   - 减少摩擦
   - 消除复杂步骤
   - 让用户轻松完成

4. **嵌入分享**
   - 在产品中自然地嵌入分享机会
   - 不是强制的推荐按钮
   - 而是自然的"告诉朋友"时刻

5. **测量和优化**
   - 测量口碑增长率
   - 识别哪些功能驱动推荐
   - 优化这些功能

### Wise 的具体战术

#### 1. 定价比较

- 展示 Wise 的费用 vs 银行的费用
- 让用户看到节省的金额
- 这个"哇"时刻驱动分享

#### 2. 速度展示

- 展示转账速度
- "你的钱将在X小时内到达"
- 超出预期创造惊喜

#### 3. 简化流程

- 减少所需的步骤
- 消除不必要的文档
- 让第一次使用就成功

#### 4. 信任信号

- 展示安全措施
- 显示监管合规
- 建立可信度

### 产品中嵌入营销

#### 实践方法：

1. **识别分享时刻**
   - 什么时候用户最想告诉朋友？
   - 通常是在成功交易后
   - 在看到节省金额时

2. **使分享变得容易**
   - 一键分享
   - 预填充消息
   - 但不是强制的

3. **激励分享**
   - 但不是通过金钱激励
   - 而是通过价值激励
   - "告诉你的朋友，让他们也能省钱"

4. **跟踪分享**
   - 测量分享率
   - 理解哪些功能驱动分享
   - 优化产品以增加分享
