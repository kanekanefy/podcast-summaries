**嘉宾简介**: Emilie Gerber - Six Eastern 公关公司创始人兼 CEO，曾在 Uber 负责商务拓展 PR、在 Box 负责产品传播。

1. **核心议题 (The Questions)**
   - **如何衡量 PR 的价值**：B2B/B2C 的“新闻价值”与增长关系到底是什么？
   - **融资/产品发布如何被媒体接住**：不同媒体（TechCrunch、Axios 等）分别吃什么类型的故事？
   - **冷启动如何触达记者与播客**：没有“关系”时怎样写出会被回复的 pitch？
   - **PR 应该外包还是自建**：何时该找机构，预算与节奏应如何设定？

2. **核心洞察 (Core Conclusions)**
   - **PR 最大价值是“第三方背书”**：对 B2B 来说，**Logo/报道像“信誉徽章”**，在招聘、销售、BD 邮件中产生二阶影响。
   - **记者不为“自夸式故事”买单**：**用“对标巨头/挑战 incumbent”建立参照系**（如 Ramp 挑战 Bill.com）比“category creation (品类创造)”更有效。
   - **冷邮件比“人脉”更关键**：**关系只带来打开率，不带来报道**；媒体更看重**清晰主题与独特观点**。
   - **三句 pitch 胜过长篇大论**：**短、直、可复用的故事钩子**最容易被记者转成标题或导语。

3. **心法与原则 (Repeatedly Emphasized Points)**
   - **“说人话”优先**：PR 文案要像口头解释一样清楚，**先回答“做什么、给谁、为什么”**。
   - **“独家 (exclusive)”是稀缺资源**：**错配独家会伤关系**，要谨慎分批投递。
   - **先找“媒体适配度”**：**媒体的栏目/人群/节奏决定你能不能上**，而不是新闻大小。

4. **实战手册 (Specific Cases & Methodologies)**
   - **融资/发布 pitch 模板**：**“金额 + 赛道 + 投资人 + 为什么重要”**直给；**3-4 天跟进**，每次补充新信息。
   - **媒体打法差异**：
     - **TechCrunch**：融资/首发；不再收 op-ed。
     - **Axios**：**偏交易/并购/融资**，不做产品发布。
     - **Business Insider**：**pitch deck 故事/创业者收入指标**。
     - **VentureBeat**：**AI 主题首选**。
     - **Fast Company**：**未来工作/设计/文化**。
     - **Forbes**：**榜单/Contributor 网络**。
   - **播客&Newsletter 寻找法**：**Google + 行业关键词** → 核对是否活跃 → 检查过往嘉宾 → **先发轻量“是否开放嘉宾”私信**。
   - **Press release 替代法**：**用创始人博客替代新闻稿**，同样信息但更可分享、更可控。

5. **反常识视角 (Counter-intuitive Points)**
   - **“人脉 PR”被高估**：**冷启动写得好比关系更有效**；媒体节奏快，旧关系难持续。
   - **“category creation (品类创造)”对 PR 反而无效**：**记者需要现成参照物**，不是新名词。
   - **“700% 增长”不等于好新闻**：**真实规模+结构性指标**比夸张百分比更可信。

6. **花絮与细节 (Interesting Details)**

   - **个人偏好**：**偏爱末日题材剧**，推荐《Yellowjackets》。
