# Gergely Orosz - 播客摘要

**嘉宾简介**: Gergely Orosz - The Pragmatic Engineer创始人，前Uber工程师经理，Substack顶级科技通讯作者

## 核心议题 (The Questions)

1. **如何从高薪技术工作转型为全职创作者？** 离开Uber年薪32-33万美元的工作，转向不确定的写作创业之路的决策过程
2. **全职newsletter生活的日常是怎样的？** 每周产出高质量内容的流程、时间管理、压力与挑战
3. **如何在订阅经济中获得成功？** 从0到189,000订阅者的增长策略、付费模式、收入天花板
4. **想要开始newsletter的人应该如何入手？** 启动策略、成功要素、常见陷阱

## 核心洞察 (Core Conclusions)

1. **意外的收入反转**：Gergely离开Uber时预期收入会大幅下降，但实际上现在通过newsletter获得的收入**超过了在Uber的最高薪**，且收入随着订阅者增长理论上没有上限

2. **"数千个微老板"**：这种生活感觉像是有数千个微老板，而不是一个老板；每天都有可能被解雇，但也每天都在获得加薪（随着订阅增长）

3. **6年意外积累的基础**：表面上看newsletter快速增长（6周内从0到1,000付费订阅者），但实际上背后有**6年的博客写作积累**。从2015年开始写The Pragmatic Engineer博客，在Hacker News等平台建立受众

4. **"渔夫的故事"哲学**：避免为了最终能够写作而辛苦工作10年（创业），而是**直接开始写作**。这是对传统职业路径的反直觉思考

5. **约束机制的重要性**：通过每周两篇的承诺给自己创造**外部压力和截止日期**，这是持续产出的关键。类似于Jeff Atwood的"每周写三次，坚持两年"建议

6. **Substack推荐引擎效应**：2021年4月引入的推荐功能成为**巨大增长引擎**，使newsletter在前9个月达到5万订阅者后，在接下来5-7个月内又增长10多万

## 心法与原则

1. **长期主义vs即时满足**："如果你写一本书，你会在4年后上市，那时可能已经忘记了你为什么要写它。但newsletter让你每周都必须创造有价值的内容"

2. **个人品牌vs公司品牌**：在一家成功公司工作会给你带来"品牌光环"，但**个人品牌是可以带走的资产**

3. **深度优先于广度**：成为某个领域的专家，然后**定期分享你的知识**。这是建立受众的关键

4. **适应性写作策略**：根据反馈和数据调整内容，但**保持核心主题一致**

## 实战手册

1. **内容生产流程**：
   - 周一：完成周二文章的最终编辑
   - 周二：发布文章，进行自由写作，为未来文章积累想法
   - 周三：自由日（但实际经常开始写周四的文章）
   - 周四：写周四的"scoop"（时效性文章）
   - 周五：开始写下周二的文章
   - 每篇文章需要：研究（1天）→ 草稿（1天）→ 反馈（1天）→ 编辑（1天）= **约4天**

2. **专注力工具**：
   - **Centered**：专注力应用，视频监控功能增加工作压力
   - **Pomodoro方法**：25分钟间隔工作
   - **脚本阻断分心**：使用Python脚本修改hosts文件，**完全阻断Twitter、LinkedIn、Facebook**
   - **20分钟启动规则**：设置20分钟计时器，**前5分钟最痛苦，之后进入状态**

3. **newsletter启动策略**：
   - **先积累内容**：在启动前已有6年博客写作经验
   - **选择合适平台**：Substack的推荐功能成为关键增长引擎
   - **付费模式**：免费+付费双层结构，付费内容更深入、更频繁
   - **6个月承诺**：给自己6个月时间全力投入，如果不成功就退款

4. **个人品牌建设**：
   - 在**顶级公司工作**（Uber、Skype等）获取"血统"认可
   - **持续公开写作**，建立专业知识
   - 在**Hacker News等平台**分享内容
   - **不要过早商业化**，先建立受众和信任

## 反常识视角

1. **收入无上限 vs 企业收入上限**：在企业工作收入有明确上限（即使是Uber），但作为创作者，**收入随着受众增长持续增长**，理论上没有天花板

2. **压力来源**：在Uber时压力来自外部（裁员、重组），作为创作者压力来自**内部期望**和**自我设定的截止日期**

3. **孤独感**：离开企业后**失去日常社交互动**，虽然通过共享工作空间补偿，但仍是一个主要挑战

4. **无法真正"退出"**：不像企业可以被收购，newsletter与个人身份深度绑定，**难以出售或退出**

5. **假期困境**：每周必须产出内容，**难以真正休假**。Gergely第一年没有休假，导致家庭摩擦

## 花絮与细节

1. **收入对比**：Uber最高年收入32-33万美元（欧洲标准），现在通过newsletter收入**超过这个数字**，且持续增长

2. **订阅者增长**：189,000订阅者，过去90天增长80,000（约**每天1,000新订阅者**）

3. **写作量**：去年写的内容相当于**4-5本书的量**

4. **工作习惯**：
   - **喜欢**：开放的日历、实验的自由、创造力的发挥
   - **不喜欢**：孤独感、持续产出的压力、无法真正休假

5. **推荐资源**：
   - **书籍**：《The Three Musketeers》（连载写作的启发）
   - **博客**：Coding Horror (Jeff Atwood)
   - **工具**：Centered、hosts文件脚本

6. **个人趣事**：
   - 最初计划离开Uber后创办创业公司，筹集风险投资
   - 意外通过电子书获得收入（第一年收入约10万美元）
   - 给自己6个月时间尝试newsletter，如果失败就退款

7. **给新手建议**：
   - **从副业开始**，不要立即全职
   - **建立专业深度**，成为某个领域的专家
   - **保持规律性**（每周或每两周）
   - **长期坚持**（Jeff Atwood的建议：每周写三次，坚持两年）
   - **不要过早商业化**，先建立受众
