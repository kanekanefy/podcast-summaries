# Sahil Mansuri - 播客摘要

## 嘉宾简介
Sahil Mansuri是Bravado的CEO和创始人,Bravado拥有世界上最大的在线销售社区,超过30万名销售人员。他本人也是资深销售人员,曾在Glassdoor负责企业销售,亲自签下了Facebook、Google、Amazon、Microsoft等大客户。

## 核心议题
1. 在市场低迷时期,创始人应该如何调整销售策略
2. 如何重新思考销售配额(comp plans)和激励机制
3. 预测与目标设定的务实方法
4. 客户留存的重要性及具体策略
5. 在困难时期提高成交率的技巧

## 核心洞察

### "漏桶"理论(The Leaky Bucket)
- 大多数SaaS公司的销售激励计划只关注新客户获取,而不考虑客户留存
- 销售代表A签了15个客户,创造了150万美元收入,获得40万美元奖金和总裁俱乐部奖励
- 销售代表B签了12个客户,创造120万美元收入,只获得25万美元
- 如果A的10个客户流失,而B的所有客户都续约并成为推荐者,A不受任何影响,B也不获得额外奖励
- 这种"Glengarry Glen Ross"式的销售文化在资本廉价时代行得通,但在当前市场是致命的

### 销售数据的真相
- 2023年Q3:63%的销售代表未达标(比Q2的54%和Q1的46%大幅上升)
- 2023年Q3:76%的公司未达标(比Q2的59%和Q1的51%上升)
- 预计Q4将有超过80%的公司错过目标
- 10月:85%的月度销售代表未达标

### "乐观偏见"的危险
- 创始人天生乐观,这在当前市场是劣势而非优势
- 9月份感觉良好,10月份却变成"大屠杀"
- 需要设置预先确定的决策点,避免在Q1错过目标时还抱有"Q2会好转"的幻想

### 文本消息vs电子邮件
- "电子邮件是交易埋葬的地方,文本消息是交易达成的地方"
- 通过文本介绍时,不要把介绍人从群聊中移除
- 保持介绍人在线程上可以增加对方参加会议的压力
- 案例通过持续9周的文本跟进最终获得了客户

### Facebook销售案例
- 发送冷邮件给Sheryl Sandberg,使用BCC尝试15个可能的邮箱地址变体
- 创建了9页报告,分析Facebook员工对公司的评价、薪资对比、Mark的评分
- 报告被添加到Facebook每周高管报告中
- 关键洞察:不是"我要卖你广告",而是"这是我发现的关于你公司的洞察"

## 心法与原则

1. **销售即教练(Coaching)**
   - 最好的销售不是让人们"感觉被销售了"
   - 而是改变他们对业务运营方式的看法
   - 成为值得信赖的顾问,而非推销员

2. **创始人必须成为销售专家**
   - CEO的工作100%是销售
   - 首先要卖给自己(创业的勇气)
   - 然后卖给人才、投资人、客户、媒体、候选人

3. **平衡悲观与乐观**
   - 伟大的销售人员内心极度悲观(不断筛选不合格的线索)
   - 但表面上保持乐观(继续向客户推销)
   - 这种双重性格是区分好销售与伟大销售的关键

4. **建立真实友谊**
   - 销售14年后,仍然可以给任何签过约的客户打电话聊天
   - 投入大量精力建立真正的友谊,而不仅仅是商业关系
   - 不做"一次性交易"的销售

## 实战手册

### 销售预测的"保守+里程碑"法
1. 制定最保守的年度计划(例如:明年收入从1000万降到900万)
2. 设置短期里程碑来解锁增长投入
   - 如果Q1达到250万,可以上调其余季度的目标
   - 如果Q1低于200万,则下调目标
3. 提前与董事会、销售团队、销售领导达成一致
4. 避免创始人天生的乐观偏见

### 重新设计销售激励计划
1. **延长激励时间线**
   - 如果客户今天签约明天续约,销售应该获得额外奖励
   - 将客户续约率纳入销售考核

2. **比较相对表现**
   - 比较不同销售代表的客户续约率
   - 奖励那些能够预先筛选合适客户的销售

3. **打破50-50分成**
   - 当前销售领导的薪酬通常50%底薪+50%提成
   - 大多数其他高管是90-10分成
   - 导致销售人员被贴上"只认钱"和"雇佣兵"的标签

4. **与客户、业务、代表的三重对齐**
   - 当前系统只对业务有利(获得新客户)
   - 应该设计同时激励长期客户关系的系统

### 客户留存的"最佳销售转CSM"策略
1. 将最好的销售代表转为客户成功经理(CSM)
   - 预售vs售后:通常把最好的人放在预售,但现在是时候逆转
   - 理由:新客户获取很难,但保留现有客户更难

2. **创建独家洞察**
   - 分析客户数据,提供行业基准报告
   - Bravado的例子:告诉客户类似公司正在招聘、调整配额、改变薪酬计划

3. **举办线下客户活动**
   - 2月举办客户专属的Napa红酒之旅
   - 条件:截至2023年2月10日仍是客户
   - 心理杠杆:人们不想失去已经获得的福利

4. **在活动中获取推荐**
   - 不要邀请潜在客户,只邀请现有客户
   - 在社交时间询问:"还有谁像你一样可能受益?"
   - 要求通过文本(而非邮件)进行介绍
   - 在介绍后的前10-20条消息中保持介绍人在群聊中

### 提高成交率的"超预期价值"策略

1. **Facebook案例的启示**
   - 不要只是推销产品
   - 创造对客户业务有独特价值的内容
   - Sahil的9页报告不是关于"为什么买Glassdoor",而是"Facebook员工如何看待公司"

2. **教学式销售**
   - 不是"我要给你价值"(很奇怪的说法)
   - 不是"我的产品能解决你的问题"(你不知道我的问题)
   - 而是"我要做一些让这段时间值得的事"

3. **利用独特视角**
   - Bravado的优势:看到50,000家公司的销售数据
   - Greenhouse/Lever的例子:告诉客户"类似Series B公司有50个员工,现在降到4个"
   - 任何供应商都有横截面数据,问题是如何提取和利用

### 创新在低迷时期

**Bravado Flex案例:**
- 问题:招聘市场放缓,没人愿意雇佣全职销售
- 解决方案:创建100%纯佣金制销售角色
- 结果:从业务严重放缓到有史以来最好的月份之一

**改变定价策略:**
- 从年费12,000改为月费1,000
- 从年费改为日费20美元
- 不是"优化"现有模式(从12降到10)
- 而是彻底改变游戏规则

## 反常识视角

1. **销售不是天赋,是可学习的技能**
   - Sahil在2008年金融危机期间开始销售
   - 2009年,在Meltwater收入从7000万降到6900万的第一年,他打破了公司单人销售记录
   - 证明即使在最困难时期,销售技能仍然可以创造奇迹

2. **最好的销售不为了钱而销售**
   - "我从未在乎过佣金"
   - 真正伟大的销售人员不是为了钱、奖杯或荣誉
   - 他们热爱销售本身,赢的快感是会上瘾的

3. **在低迷时期仍然要增长**
   - "公司要么增长,要么死亡,没有中间状态"
   - 只是试图"挺过冬天"最终会失败
   - 员工会士气低落,投资者会失去信心,时间会变得漫长

4. **销售多样性问题**
   - 92%的销售领导是白人
   - 85%以上的销售领导是白人男性
   - 不是因为白人男性更适合做销售领导
   - 而是"我们不想要创新,只要雇佣做过这事的人"

5. **短期优化是长期问题的根源**
   - 选项A:本季度收入增长75%,但不承诺未来
   - 选项B:未来6个季度每季度增长20%
   - 6个月前,99%的创始人会选择A
   - 现在所有人都为选择A付出代价
