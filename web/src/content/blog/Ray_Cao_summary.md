# Ray Cao - Podcast Summary

## 嘉宾简介

Ray Cao是TikTok全球货币化产品战略与运营负责人，曾在Google工作6年帮助Google Shopping全球化，领导了TikTok广告业务从2人团队到全球扩张的快速增长。

## 核心议题

1. TikTok的核心文化价值观和运营方式
2. TikTok vs Google在文化、创新、产品开发上的差异
3. 如何在TikTok上成功做广告和内容
4. TikTok的产品组织结构和快速创新机制
5. 全球化产品的本地化策略

## 核心洞察

**比喻/英文术语:**

- **情境而非控制(Context, No Control)** - 给员工完整信息背景，不具体控制，让他们像业务主人一样思考和连接所有点
- **主动思考，反应行动(Proactive Thinking, Reactive Doing)** - 先主动思考，然后在行动时主动协作
- **马拉松第二半程更快** - "How can I run my second half of my marathon faster than the first half?" - 保持创业公司饥饿感
- **内容图谱vs人脉图谱/意图图谱** - TikTok是基于内容(content graph)，Meta是基于人(people graph)，Google是基于意图(intent graph)
- **探索与利用(Explore and Exploit)** - TikTok不断尝试新内容看什么能火
- **口语化的数字版(word-of-mouth的数字版)** - TikTok就像数字化的口碑传播，帮助用户发现新事物
- **工程师-产品-销售紧密协作** - TikTok的独特优势是这三个团队比其他公司更紧密地一起工作

## 心法与原则

1. **始终Day One** - 保持创业公司心态，总是饥饿成长，避免陷入"我们已经成功了"的陷阱
2. **信息透明+完整背景** - 给所有人完整情境，让他们能看到拼图的全貌，而不是只看自己那块
3. **全球人才本地化** - 真正全球化需要深入当地市场，雇佣本地人才，理解当地文化
4. **工程师接触市场** - 定期带PM和工程师见客户，让他们"感受热度"和一线痛点
5. **质量优于数量** - 快速招聘时不能降低标准，错误的人会拖慢而不是加速
6. **主动思考必须协作** - 有了主动思考和情境后，行动时必须主动协作，不要越界
7. **读写会(Meeting in Reading Format)** - 180人的大会议，先读文档再评论讨论，保持全员共识

## 实战手册

**TikTok广告成功策略:**

1. **平台差异化** - 不要用Instagram或Google的逻辑套用到TikTok
2. **创意测试** - 每周至少测试10个不同创意素材
3. **广泛定向** - 早期避免过于细分的定向，让算法找到合适受众
4. **先有机后付费** - 先建立有机业务账号感受平台，再启动广告
5. **拥抱平台文化** - 创作TikTok原生内容，不是直接搬运其他平台素材
6. **至少尝试一个月** - 给自己和平台时间学习曲线
7. **利用工具** - 使用CapCut等工具快速创作内容

**内容创作成功要素:**

1. **真实不做作** - 不需要完美，做真实的自己
2. **持续测试** - 不断尝试新内容找到竞争优势
3. **拥抱文化和社区** - 理解平台用户行为和喜好
4. **与趋势连接** - 关注平台趋势，将广告与流行元素结合

**产品团队组织:**

1. **全球部署** - 在关键市场设立工程和PM团队（新加坡、纽约、洛杉矶等）
2. **频繁重组** - 根据市场需要快速调整团队结构，不被既定结构束缚
3. **180人读写会** - 每季度把工程、产品、销售核心团队聚在一起，先读文档再讨论
4. **市场沉浸** - 定期带PM和工程师参加客户会议，感受一线
5. **OKR聚焦** - 使用OKR系统，重点是跨功能对齐，而不是各自为政

## 反常识视角

1. **TikTok不是只靠算法** - 虽然算法很重要，但仍需要大量本地人才调优，理解不同市场文化
2. **大公司也可以快速创新** - TikTok通过特定组织方式保持创业速度，包括读写会、紧密跨职能协作、频繁重组
3. **内容差异化很小但效果差异很大** - 同样的内容在Instagram和TikTok表现完全不同，需要平台原生创作
4. **目标太多=没有目标** - 当你有17个优先级时，你实际上没有优先级
5. **工作辛苦是一种选择** - 加入创业公司是生活方式选择，不是9-5工作，不适合所有人
6. **离职可以庆祝** - 如果员工去更好的地方成长，应该为他们高兴，保持这种文化

## 花絮与细节

- TikTok工程师-产品-销售团队的距离比其他公司近得多，这是秘密武器之一
- 泰国墨盒案例：Premium产品卖不动是因为消费者不在乎打印质量，只在乎耐用性
- 在日本，TikTok上美食内容比唱歌跳舞更流行
- 东南亚流行3C电子消费品内容
- Ray的第一个工作是做东南亚市场进入研究，花了1年半才理解不同市场的文化差异
- TikTok在美国先推购物，在东南亚先推创作者基金，在欧洲推游戏广告 - 不同市场不同策略
- 前6个月需要招聘100人支持全球GTM，这是Ray犯的最大错误：为了速度降低质量标准
- Ray从Facebook带来了"context, no control"文化，但起初试图创造严格纪律，这拖慢了团队
- TikTok从Amazon学到了"Always Day One"和读写会文化
- TikTok从Google学到了OKR系统
- TikTok推荐测试至少每周10个创意，"The more, the better"

- Ray和Lenny都在Duolingo的Emerald联盟
