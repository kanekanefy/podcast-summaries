# Jerry Colonna - 成长领导力艺术

**嘉宾简介**: Jerry Colonna - Reboot联合创始人CEO,全球最受尊敬的高管教练之一,曾任Flatiron Partners联合创始人(与Fred Wilson),JPMorgan Chase合伙人,著《Reboot》和《Reunion》。

## 1. 核心议题 (The Questions)

- **为什么成功不一定带来幸福?** - 社会化大谎:成功、金钱、大房子不等于快乐
- **如何在成就中不杀自己?** - 觉察、成长练习、自我同情
- **什么真正摧毁团队和公司?** - 不是缺乏人才/战略/执行,而是未解决的童年恶魔
- **如何在AI时代保持人性?** - 存在、连接、战略思考被提升

## 2. 核心洞察 (Core Consequences)

### 颠覆性认知
- **"不必成为混蛋也能成功"** - 19岁孩子总结,不必为事业感到痛苦
- **"增长思维的危险"** - 把概念变成固定思维,"这是好/坏"二元分类
- **"忙碌的上瘾"** - 说不想忙,但日程不塞满感到不安
- **"成功不代表配得上爱"** - 不需要A+拼写测试才被爱

### 核心方程
**Practical Skills + Radical Self-Inquiry + Shared Experiences = Enhanced Leadership + Greater Resilience**

(实用技能 + 激进自我询问 + 共同体验 = 增强领导力 + 更大韧性)

### 关键概念
- **Radical Self-Inquiry (激进自我询问)** - 切穿自我妄想的问题
- **Complicit (共谋)** - 不是"负责",是"从犯"(开车接应不是抢银行)
- **Attachment (执着)** - 执着结果无意中助长自己痛苦
- **Spiritual Bypassing (灵性逃避)** - 去秘鲁喝ayahuasca,周末用迷幻药,不面对需要照顾的部分

### Lemon Drops故事
- **童年稳定**: 祖父Grandpa Guido无尽柠檬糖供应
- **柠檬糖=财富**: 稳定感和财富概念匹配
- **30多岁成功**: 有柠檬糖但不安全
- **核心问题**: 我与金钱关系如何塑造职业/学校/安全/自我价值选择?

### 四圣谛应用
1. **人生充满痛苦** - 出生、老、病、死
2. **推走痛苦增加痛苦** - 买大房子/沙发为推开"我够好被爱吗?"痛苦
3. **有痛苦的尽头** - 重要真理
4. **八正道** - 痛苦终结之路

## 3. 心法与原则

### 佛教智慧
- **自我妄想+执着=我们痛苦最大贡献者**
- **一切总在崩解**: 即使增长思维也崩解
- **中等道路**: 执着成长,松散持守心态

### 成长练习不是科学时刻
- **不是一天醒来完成**
- **持续练习**: 不是持续改进,而是持续询问
- **可能感觉疲惫**: 思考时,但生活时激活

### 领导者责任
- **权力越大道德责任越大**: 暂停搞清你在做什么共谋创造不想要条件
- **从最多权力人开始**: 如果那个人拒绝做工作,整个组变成早期不完美体现
- **不检视生活请不要做涉及他人工作** - Parker Palmer建立Socrates

### 团队动态
- **替罪羊 (Scapegoat)**
- **说真话的人 (Truth-teller)** - "让我告诉你真正错在哪"
- **重复原生家庭模式**: 没有个人激进自我询问技能

### Carl Jung引用
**"直到你让无意识意识化,它将指导你生活而你称其为命运"**

应用团队:直到让组内无意识模式意识化,组将继续重复那些模式而你怪组内某人

### Unexamined Life
- **Socrates**: "不检视的生活不值得过"
- **Parker Palmer玩笑**: "但如果你选择不检视生活,请不要做涉及他人工作"

## 4. 实战手册

### 激进自我询问问题

**核心问题**:
1. **"我如何在我说我不要的条件中成为共谋?"**
   - 共谋≠负责
   - 从犯(accomplice):开逃跑车不是抢银行出纳
   - 目的:唤起你自己能动性
   - 例子:"我不想总是忙,但我的日程不塞满我感到不安不安"

2. **"我没说什么我需要说?"**
   - 适用于关系
   - 也适用于领导他人

3. **"我在说什么没被听到?"**

4. **"什么在被说而我没听到?"**

**当你处于激进自我询问区域**:
- 问题让你呼吸停止
- 你不必与任何人分享答案(除了自己)
- 与朋友/同事/教练/治疗师分享可能有力量
- 最重要人:自己

**其他问题**:
- "这会为你做什么?"(关于野心/目标)
- "你如何定义成功?"
- "那从哪来?"
- "没有你的故事你会是谁?"
- "你希望生活人知道你什么但你太害怕告诉他们?"
- "我日结束时我想后来人说关于我什么?"
- "我想成为什么样的祖先?"
