# Jeffrey Pfeffer - 权力七大法则

**嘉宾简介**: Jeffrey Pfeffer - 斯坦福商学院组织行为学教授,讲授最受欢迎的课程之一《权力之路》,著有多本关于权力的书籍。

## 1. 核心议题 (The Questions)

- **为什么获取权力让人不舒服?** - 权力的现实与道德教育、家庭价值观相悖,人们将工具与滥用者混淆
- **如何在职业中建立和运用权力?** - 通过七大可学习的技能系统性地构建影响力
- **为什么"好人"更需要权力?** - "如果你想让权力被用于善,更多好人需要拥有权力"
- **个人品牌和社交网络如何转化为职业权力?** - 可见性与实质能力同样重要

## 2. 核心洞察 (Core Conclusions)

### 颠覆性认知
- **权力是中性的工具** - "我见过有人用锤子打人,但这不代表锤子不是有用工具" - 外科医生Laura用手术刀治愈癌症,劫匪用刀抢劫
- **权力不是终点而是手段** - 你可以选择权力或自主,但不能兼得 - "我可以给你十亿美元,但你买不到上周"
- **世界不是自动平衡的** - Icarus神话(飞得越高摔得越惨)是错的,成功会带来更多成功:"生活是自我实现的"

### 关键比喻
- **"做权力"(Doing Power)** - 课程要求学生实践权力原则,因为不用就忘,如学法语必须说话
- **"海水里的渗透压"** - Laura Esserman用足够能力克服摩擦力,减少摩擦力能成就更多
- **"台前表演"(Kayfabe)** - 组织是剧场,你需要扮演好角色而非暴露真实自我

### 研究支撑
- Gerald Ferris的政治技能量表:政治技能与**薪水、晋升、职业幸福感、低压力**正相关
- Mark Granovetter的《弱关系的力量》:最佳工作来自"弱关系"而非密友
- 目标设定文献:设定目标显著提高达成概率

## 3. 心法与原则

### 底层哲学
- **"如果我想被喜欢,我就去养狗"** (Gary Loveman,前哈拉斯CEO)
- **"保持好奇,不要评判"** (Walt Whitman)
- **"评判他人时就没有时间去爱他们"** (Mother Teresa)

### 权力七大法则
1. **别挡自己的路** - 克服冒名顶替综合症,停止预先道歉
2. **打破规则** - 规则由既得利益者制定,通过差异化让自己被记住
3. **展现强大** - 外表、声音、肢体语言比内容更重要
4. **建立强力品牌** - "他们不能选择脑子里没有的人"
5. **无情社交** - 成为连接者, generosity是首要原则
6. **使用你的权力** - 权力像肌肉,越用越强
7. **成功会开脱一切** - "forgiven, forgotten, or both"

## 4. 实战手册

### 个人品牌建设
- **Laura Chau案例** (4年成为Canaan合伙人):
  - 写文章、开播客邀请有影响力的人、出书、组织社交晚宴
  - 利用身高优势(穿高跟鞋到6英尺1英寸)、独特穿衣风格
  - 战略性思考"如何让人知道我是谁"

### 社交网络策略
- **列出10个关键人物** - 如果认识他们会对你职业生涯重要,制定接触计划
- **成为broker** - 连接不同群体的人,风险投资商连接想法与资金并收费
- **追求弱关系** - 强关系圈信息冗余,弱关系带来非冗余信息和新人脉

### 表现力量技巧
- **眼神交流** - Tony Hayward因念稿无法眼神接触显得虚伪
- **不读笔记** - Jack Valenti(38年MPA主席)"从不带笔记见国会"
- **愤怒信号** - 打破规范显示权力
- **更大音量、更多手势、开放式姿态**
- **侵入个人空间** - 轻触手臂、肩膀、握手

### 作业设计
1. **目标设定** - "10周后成功是什么样子?"
2. **权力属性评估** - 评估精力、专注度、容忍冲突等7个维度
3. **资源创造** - 学生创立奖项、播客等活动
4. **网络建设** - "列出10个想认识的人并制定接触策略"
5. **品牌陈述** - "你想如何被认知?"给MBA求职者用
6. **力量表演** - 录制60-90秒视频,扮演Tony Hayward但做得更好

### 15%渐进突破
- 类似Touchy-Feely课程的15%规则
- "不要让明天做太离谱的事,从舒适区外推一点,再推一点"
- 如游泳英吉利海峡,从泳池开始每天增加难度

## 5. 反常识视角

### 颠覆常识的发现
- **别想被喜欢** - "你被雇佣是做事,不是人可爱" - 选外科医生技术优先于性格
- **停止道歉** - "预先道歉、'原谅我占用时间' - 如果评论没用就别说"
- **主动求助** - Frank Flynn研究:人们高估需要求助的人数,求助让人不适但有效
- **不做工作去社交** - Omid Kordestani放弃工作专注内部社交和外部关系,成为Google第11名员工赚25亿

### Omid Kordestani案例
- 1995年在Netscape营销职位无进展
- **决定停止做工作**,专注内部社交网络
- 与高管关系好后,"他们不太在乎我做不做工作"
- 开车遍访硅谷,认识所有人
- 1998年Google寻找第一名商务人员,Omid在所有名单上

### Jason Calacanis案例
- 违反VC行业常规:做很多小额投资而非几个大额
- 无合伙人的精简运营(不能被合伙人解雇)
- 新闻记者背景让他不断提问聪明人
- "几乎没人喜欢他,但这没关系"(权力法则一)
